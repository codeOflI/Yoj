# yoj
online judge

**此为后端开发环境。后端开发工具：idea**

**前端开发环境仓库**：[**yoj_front_end**](https://github.com/Li-MingZhong/yoj_front_end)      **前端开发工具：vs code**

**前端：vue+vuex+vuecli+vue-router+bootstrap(jquery)+axios**

**后端：spring boot+ mybatis + spring security+redis**

**模板引擎:thymeleaf**
